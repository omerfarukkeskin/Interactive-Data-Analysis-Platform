# Interactive Data Analysis Platform
It is an interactive, Streamlit-based web application that allows users to manage end-to-end machine learning processes by uploading CSV files. Data preprocessing, target variable selection, preprocessing, model training, hyperparameter optimization, performance evaluation, and visualization are all performed.



## Supported Models
### Classification
* Logistic Regression
* K-Nearest Neighbors
* Random Forest
* Gradient Boosting

### Regression
* Multiple Linear Regression
* Random Forest Regressor
* Gradient Boosting Regressor
* XGBoost Regressor

## Evaluation Metrics
### Classification

* Precision

* Average Precision

* ROC CUrve
* Precision-Recall Curve

  ### Regression
